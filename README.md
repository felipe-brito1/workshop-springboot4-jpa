# Descrição:
API REST desenvolvida com Spring Boot para cadastro e listagem de usuários com PostgreSQL, utilizando JPA/Hibernate e deploy na nuvem com Render.
# Como acessar:
Clone o repositório com git clone https://github.com/felipe-brito1/workshop-springboot4-jpa.git, acesse a pasta com cd workshop-springboot4-jpa, configure o banco no application-dev.properties com suas credenciais 
PostgreSQL, execute com ./mvnw spring-boot:run e acesse em http://localhost:8080 ou utilize a versão online em http://workshop-springboot4-jpa-08t6.onrender.com/
